# 餐饮管理系统简介 java63

## 项目概述

餐饮管理系统是一款基于Spring Boot与Vue的前后端分离的餐饮业务管理软件。它集成了多种功能模块，旨在提高餐饮业务的工作效率和经营管理水平。

## 技术栈

- 开发环境：Windows 10/11
- 架构模式：MVC、前后端分离
- JDK版本：Java JDK 1.8
- 开发工具：IDEA
- 数据库版本：MySQL 5.7以上
- 数据库可视化工具：Navicat
- 服务器：Spring Boot自带的Apache Tomcat

主要技术：

- 后端：Java、Spring Boot、MyBatis、MySQL
- 前端：jQuery、HTML、Vue

## 功能模块

- 个人中心
- 管理员管理
- 菜品管理
- 论坛管理
- 供应商管理
- 公告信息管理
- 单页数据管理
- 用户管理
- 员工管理

## 系统角色

- 管理员
- 用户

## 运行环境

- 服务器：Apache Tomcat
- 数据库：MySQL 5.7以上
- 客户端：现代浏览器

## 核心亮点

- 基于Spring Boot的微服务架构，便于维护和扩展。
- 采用MVC模式，使得业务逻辑与用户界面分离，便于管理。
- 集成多种管理模块，满足不同角色的业务需求。
